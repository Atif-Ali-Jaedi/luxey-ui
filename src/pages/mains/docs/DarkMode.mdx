# Dark Mode

Enabling Dark Mode in your projects is straightforward:

## Using CDN

<Timeline
  data={[
    {
      id: 1,
      heading: "Update the library",
      text: "Ensure you are using the latest version of our UI Library.",
      code: {
        lang: "html",
        code: `<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/luxeyui@${currentVersion}/dist/all.min.css" />
<script src="https://cdn.jsdelivr.net/npm/luxeyui@${currentVersion}/dist/all.min.js" defer></script>`,
      },
    },
    {
      id: 2,
      heading: "Enable Dark Mode",
      html: `To enable dark mode, please add thhis code <code class="code before:content-[''] after:content-['']">darkMode: 'class'</code> in your <code class="code before:content-[''] after:content-['']">tailwind.config.js</code> file.`,
      code: {
        lang: "js",
        code: `tailwind.config = { darkMode: "class" }`,
      },
    },
    {
      id: 3,
      heading: "Customize as Needed",
      html: `Modify your css file to tweak the Dark Mode to suit your design needs. An example given below. Make sure to use <code class="code">hsla</code> format and without the parenthesis and <code class="code">hsla</code> to customize the colors.`,
      code: {
        lang: "css",
        code: `:root.dark {
	--base-primary: 188.6 100% 46.7%;
	--content-4: 0 0% 99.6%;
}`,
      },
    },
    {
      id: 4,
      heading: "Add the toggle functionality",
      text: "Now all you have to do is to make a functionality for toggling dark mode. For example:",
      code: {
        lang: "html",
        code: `<input type="checkbox" id="toggler" class="switch" />
<script>
  document
		.querySelector("#toggler")
		.addEventListener(
			"click",
			e =>
				(document.documentElement.className = e.target.checked
					? "dark"
					: "")
		);
</script>`,
      },
    },
  ]}
/>
