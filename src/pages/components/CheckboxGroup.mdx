# Checkbox Group

A Checkbox Group allows users to select one or more items from a list of choices.

## Slots

- **checkbox-group**: Checkbox Group root wrapper which wraps `checkboxes`, `checkbox-group-label`, `description` and `error-msg`.
- **checkbox-group-label**: Checkbox group label, it is placed before the wrapper.
- **checkboxes**: Checkbox group wrapper, it wraps all checkboxes.
- **description**: The description of the checkbox group.
- **error-msg**: The error message of the checkbox group.
