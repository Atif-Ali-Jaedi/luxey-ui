# Input

A component that allows users to enter text or numbers or other characters.

## Slots

- **input-base**: Input wrapper, it handles alignment, placement, and general appearance.
- **input-label**: Label of the input, it is the one that is displayed above, inside or left of the input.
- **input-main-wrapper**: Wraps the `input-wrapper` when position of the label is `outside` or `outside-left`.
- **input-wrapper**: Wraps the `input-label` (when it is inside) and the `inner-wrapper`.
- **inner-wrapper**: Wraps the `input` and the content before / after the `input`.
- **input**: The input element.
- **helper-wrapper**: Wraps the `description` and the `error-msg`.
- **description**: The description od the input.
- **error-msg**: The error message of the input.
