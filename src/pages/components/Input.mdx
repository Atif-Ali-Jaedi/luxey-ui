# Input

A component that allows users to enter text or numbers or other characters.

## Slots

- <HL>input-base</HL>: Input wrapper, it handles alignment, placement, and general appearance.
- <HL>input-label</HL>: Label of the input, it is the one that is displayed above, inside or left of the input. 
- <HL>input-main-wrapper</HL>: Wraps the `input-wrapper` when position of the label is `outside` or `outside-left`.
- <HL>input-wrapper</HL>: Wraps the `input-label` (when it is inside) and the `inner-wrapper`.
- <HL>inner-wrapper</HL>: Wraps the `input` and the content before / after the `input`.
- <HL>input</HL>: The input element.
